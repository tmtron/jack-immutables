# jack-immutables
Testing the android jack compiler with Immutables.org

There are currently problems when building the project in Android Studio 2.2.1.

The project is a standard (almost) empty Android project with only one activity.  
The project uses the Android Jack compiler together with some annotation-processing.

there are 3 branches:
 * `android-annotations`: uses only Android-Annotations
 * `master`: uses Android-Annotations and immutables.org annotation processing
 * `generatedCode`: uses only Android-Annotations - but also contains the file `ImmutableSimpleData` which was generated by immutables.org
 
 The problem can be seen in `master` branch:
  * using `Run` - `Rebuild Project` works
  * then make any change (e.g. alter a fixed string)
  * use `Run` - `Make Project` fails with this error:
